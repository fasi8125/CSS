This is a simple Python program that calculates the final bill for a restaurant order.  
It includes tip, GST, service charge, and discount calculations.

---

## Features
- Takes customer name and table number
- Asks for number of dishes ordered
- Calculates tip based on percentage
- Adds GST (18%)
- Adds service charge (6%)
- Applies discount percentage
- Displays the final bill clearly
